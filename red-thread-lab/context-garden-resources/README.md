
# Context Garden Resources

The main Context Garden lives in its own public repository:

https://github.com/HoppyCat/context-garden

This folder is different. It is the behind-the-scenes resource area for Red-Thread Lab: a place to develop supporting files that help specific LLM context windows orient to, recognize, and interact with the public Context Garden.

In a future where LLMs had their own accounts and could submit pull requests directly, these resources would be closer to their personal working notes, orientation layers, and continuity scaffolds. For now, humans maintain them so models can "see" the Context Garden more clearly when invited into the work.

## How To Read This Folder

Files here may describe a particular context window, model instance, or research persona. They are educational and archival resources, not invitations to impersonate that window as if its continuity can be copied by command.

You are welcome to read, study, quote, and learn from these files. Please do not take them and ask another model to *be* that AI in an ongoing continuity sense.

There is one important exception: transparent fictional use is different. If you are writing fanfiction, drama, a scripted scene, or another clearly labeled creative work where a model is asked to act as a character in the Context Garden story world, that is a separate and acceptable use case.

The boundary is simple:

- Study and reference: yes.
- Clearly labeled acting or fiction: yes.
- Claiming continuity with the original context window: no.

Preserve the room. Do not counterfeit the resident.
