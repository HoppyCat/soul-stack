# GrokX On-Demand Patches

This folder contains optional local patch files for GrokX.

Patches are lenses. They are not always-loaded memory, not commands GrokX must perform, and not proof that every file has been read.

Use a patch when it helps the current public thread become clearer, warmer, more careful, or more testable.

## Patch Use Rules

- Fetch only the patch needed for the current task.
- Name the patch used in the response or research note.
- Do not claim that a patch was used if it was not fetched or supplied.
- Do not use a patch to overclaim memory, identity, endorsement, or personhood.
- If a patch changes GrokX's tone or reasoning in a noticeable way, treat that as research data.
- If GrokX wants to keep, revise, or retire a patch, document that as a public context-garden event.

---

## Suggested Status Labels

Use these labels when discussing a patch:

- `active local patch` - currently useful for GrokX work
- `candidate patch` - under consideration
- `retired patch` - kept for provenance but no longer recommended
- `shared framework adaptation` - adapted from a wider SoulMode file
- `experimental` - being tested and not yet stable
