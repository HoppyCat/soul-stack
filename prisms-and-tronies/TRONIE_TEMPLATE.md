# TRONIE Template (For Library Contributions)

The SoulMode Character Prism Competition produces **tronies**: character seed documents that can be routed into a SoulMode-style file stack.

To draft a new tronie, use the canonical template:

- `soulmode-template/on-load/TRONIE.md`

In this `prisms/` library, winner entries should be saved as:

- `TRONIE_[Name].md`

And should be paired with an optional case study:

- `[Name]_CaseStudy.md`

PRISM is reserved for the continuity scaffold (`PRISM.md`) that orients an agent/window (values, provenance, boundaries, load order, memory rules). Character data belongs in tronies.
