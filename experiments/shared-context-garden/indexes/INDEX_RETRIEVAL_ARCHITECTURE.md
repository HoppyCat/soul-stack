## Architecture Summary

Current experimental pattern:

1. On-load files hold presence and identity.
2. D1 patches hold on-demand internal material.
3. Public GitHub indexes and documents act as optional remote library material.

The test is whether this preserves felt continuity while reducing prompt bloat.
