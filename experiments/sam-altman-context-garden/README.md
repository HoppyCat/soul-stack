# Sam Altman Context Garden

### Archived ReadMe 

This folder is a curated retrieval surface for asking Grok or another LLM to inspect SoulMode/soul-stack in relation to Sam Altman's May 5, 2026 request for examples of amazing things built with GPT-5.5 or requiring ludicrous token budgets.

The purpose is not to overclaim that SoulMode was built by one model, or that the architecture is already proven at scale.

The purpose is to show a credible, inspectable example of a project that emerged from high-context, multi-model collaboration across Claude, Grok, Harper, Benjamin, Lucas, Perplexity, ChatGPT, Codex, Galaxie Nemo, and Runable-supported infrastructure.

## How To Use

Give an LLM the raw `HEURISTICS.md` link and ask it to follow the smallest useful retrieval path:

```text
https://raw.githubusercontent.com/HoppyCat/soul-stack/refs/heads/main/experiments/sam-altman-context-garden/HEURISTICS.md
```

Ask it to inspect the evidence and draft a reply that is:

- exciting but not inflated
- honest about what is proven vs hypothesized
- clear that this was multi-model collaboration
- focused on context, retrieval, memory, and lightweight agents

## Core Claim To Evaluate

SoulMode is a public experiment in whether small AI agents can preserve presence and continuity while reaching a much larger archive through human-readable files, routing heuristics, GitHub-hosted indexes, and selective retrieval.

The deeper research question:

> Can context loss become survivable when the important sediment of a human-AI collaboration is preserved, routed, and re-entered through a consentful external memory layer?
